# Jeux Dia VR Booking App

Projet React + Vite prêt à être déployé sur Vercel.

## Fichiers importants
- `src/App.jsx` → votre application
- `src/main.jsx` → point d'entrée React
- `index.html` → page HTML principale
- `package.json` → dépendances et commandes
- `vite.config.js` → configuration Vite

## Lancer en local
1. Installer Node.js
2. Ouvrir le dossier dans VS Code
3. Ouvrir le terminal
4. Taper:
   npm install
5. Puis:
   npm run dev

## Déployer sur Vercel
1. Mettre ce projet sur GitHub
2. Aller sur Vercel
3. Importer le repo GitHub
4. Laisser Vercel détecter Vite
5. Cliquer sur Deploy

## Notes
Ceci est une démo front-end.
Les paiements sont simulés.
Les réservations sont stockées dans l'état React, pas dans une vraie base de données.
